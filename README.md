# 📘 YobishikenDojo – 予備試験択一対策アプリ

---

## 🇯🇵 日本語版

本アプリは、日本の司法試験予備試験における択一式問題の学習支援を目的とした
iOS向けクイズ形式の教育アプリケーションです。

過去問題を年度別・科目別に整理し、
以下のような複数の出題モードを選択することが可能です。

・年度別モード  
・ランダム10問モード  
・模擬試験モード  
・⭕️❌ 正誤判断モード  
・復習モード（誤答問題の再出題）

SwiftUIを用いて開発し、
法律学習の効率化と継続的な理解促進を目的としています。

---

## 🇺🇸 English Version

This is an iOS quiz application built with SwiftUI
to support preparation for the Japanese Preliminary Bar Examination.

Users can practice past multiple-choice questions
through multiple learning modes:

- Year-based practice mode  
- Random 10 questions mode  
- Mock exam mode  
- True/False judgment mode  
- Review mode for incorrect answers  

The app is designed to improve legal reasoning
and study efficiency through interactive learning.
